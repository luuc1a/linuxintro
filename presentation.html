Ein paar grundlegende Befehle in der Shell

oder How to move and edit

mkdir (make directories)

Einen (oder mehrere) neue Ordner/Folder/Verzeichnisse anlegen, sofern sie nicht bereits exisiteren.
$ mkdir neuerOrdner

touch

"Berühre" eine Datei - um sie entweder zu erzeugen oder den timestamp des letzten Zugriffs zu aktualisieren.
$ touch neueDatei

cp (copy)

Eine Datei kopieren.
$ cp original kopie

mv (move)

Verschiebe eine Datei indem du sie umbenennst.
$ mv alterName neuerName.txt

$ mv datei.txt Downloads/

rm (remove)

Eine Datei löschen.

$ rm textdatei1.txt

-r recursive

$ rm -r testordner

rmdir ("remove directories")

Dieser Befehl löscht Verzeichnisse. Allerdings nur wenn sie leer sind.

$ rmdir leererOrdner

* (wildcard)

Platzhalter für beliebige Zeichen.

$ mv * ordner/

Textdateien schreiben & lesen

nano

Den Inhalt eines Files bearbeiten / Text reinschreiben.

$ nano erste-datei.txt

^X --> [Strg + x] drücken

cat

Den Inhalt eines (oder mehrerer) Files ausgeben.

$ cat erste-datei.txt

^X --> [Strg + x] drücken

"concatenate files and print on the standard output"
"concatenate" heißt "aneinander ketten".
$ cat erste-datei.txt erste-datei.txt erste-datei.txt erste-datei.txt

man     -- zeigt manual zum command
